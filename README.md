📌 Project Overview

Lensisy is a modern, responsive photography website designed to showcase services like wedding photography, cinematography, and product shoots.
This project focuses on frontend development while aligning with my goal of building Apps Script-based web applications.

🎯 Objective

The main goal of this project was to:
Strengthen frontend skills (HTML, CSS, JavaScript)
Understand UI/UX for real-world applications
Prepare for building Google Apps Script Web Apps
